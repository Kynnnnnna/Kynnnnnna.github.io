Map - ECE297 Software Design and Communication

A worldwide map application with search, routing, and high-performance rendering.

Overview

EduSpot is a Google Maps–like GIS application that supports worldwide map exploration, location search, and route planning. Users can pan and zoom across large geographic regions, search for streets and intersections, and compute routes between locations using optimized pathfinding algorithms.

The system is designed with performance in mind. A tiling-based rendering pipeline enables fast map loading and smooth interaction over large datasets, while efficient routing is achieved using A* and Dijkstra’s algorithm. These optimizations allow the application to remain responsive even in dense urban regions and complex routing scenarios.

To support multi-stop routing and tour planning, the project explores multi-destination optimization techniques inspired by the Traveling Salesman Problem. Heuristic search, simulated annealing, and reinforcement learning are used to improve route quality and avoid poor local minima in complex cases.
