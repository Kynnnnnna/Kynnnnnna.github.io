Multi User Chat System - ECE361 Computer Networks

Terminal-based real-time chat application using TCP sockets and multithreading in C.

Overview

This project implements a full client–server chat system in C using TCP sockets and POSIX threads. Users can authenticate, create or join chat sessions, send real-time messages, switch between sessions, and invite other users to sessions. The server handles multiple concurrent clients and manages shared state using mutex-protected data structures.
