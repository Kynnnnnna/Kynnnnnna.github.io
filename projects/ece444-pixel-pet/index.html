Pixel Pet - ECE444 Software Engineering

A gamified task manager where completing real-world tasks helps your virtual pet grow and evolve.

Overview

Pixel Pet is a Tamagotchi-inspired productivity web application designed to make task management more engaging and motivating. Instead of static to-do lists, users care for a virtual pet whose growth, mood, and progression reflect their real-world productivity. Completing tasks, maintaining streaks, and achieving milestones directly influence the pet’s development, turning everyday work into a playful, rewarding experience.

The system is built as a microservices-based web application. A React + Vite frontend communicates with a Flask-based API Gateway, which routes requests to independent backend services for user management, tasks, pet state, and activity tracking. Services are containerized with Docker to support local development and scalable deployment.
